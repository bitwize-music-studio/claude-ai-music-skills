- Full-text searchable

- https://news.google.com/newspapers

**Newspapers.com**

- Library access typically required

**NewspaperArchive.com**

- Subscription required

## Finding Contemporary Coverage

### Why Contemporary Sources Matter
- Written before outcome known
- Captures uncertainty of the moment
- Different framing than retrospective
- Language of the era

### Search Strategies

**By date range**:
```
"[event]" newspaper [month] [year]
"[event]" site:newspapers.com
"[event]" site:chroniclingamerica.loc.gov
```

**For specific publications**:
```
"[event]" site:nytimes.com before:1950
"[event]" "New York Times" [year]
```

### Reading Historical News
- Note sensationalism of era
- Watch for bias (political, regional)
- Cross-reference multiple papers
- Check for corrections/follow-ups

## Working with Historical Documents

### Source Evaluation
- **Provenance**: Where did this come from?
- **Authenticity**: Is it what it claims to be?
- **Perspective**: Who created it, why?
- **Corroboration**: Do other sources confirm?

### Dating Documents
- Publication date vs. event date
- Consider when author wrote (perspective)
- Note if retrospective account

### Citing Historical Sources
```markdown
**Source**: [Archive name]
**Title**: "[Document title]"
**Date**: [Original date]
**Collection**: [Collection name]
**URL/Location**: [How to find it]
**Accessed**: [When you accessed]
```

## Era-Specific Tips

| Era | Primary Sources | Challenges |
|-----|-----------------|------------|
| Pre-1860 | Letters, official records | Limited survival |
| 1860-1920 | Newspapers, photographs | Bias, sensationalism |
| 1920-1950 | Radio, newsreels, print | Audio/video rare |
| 1950-1990 | TV, magazines, gov docs | Copyright restrictions |
| 1990-2010 | Early web, digitized print | Dead links, format issues |

## For Lyrics

### Period Language
Research the language of the era:
- Newspaper headlines
- Contemporary speeches
- Slang dictionaries
- Period fiction

### Authentic Details
- Exact dates from contemporary sources
- Names/titles as used at the time
- Dollar amounts (note inflation)
- Place names (may have changed)
