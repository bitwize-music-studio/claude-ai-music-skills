# Hip-Hop

## Genre Overview

Hip-hop is characterized by rhythmic vocal delivery (rapping), beats, and sampling. From boom-bap to trap, from conscious to party, the genre spans an enormous range unified by rhythmic flow, lyrical focus, and beat-driven production.

## Characteristics

- **Vocals**: Rapping, rhythmic spoken delivery, varies from smooth to aggressive
- **Production**: Beat-driven - samples, drum machines, 808s, synthesizers
- **Lyrics**: Ranges from braggadocious to conscious, storytelling to party
- **Flow**: Rhythmic patterns, rhyme schemes, cadence variation
- **Structure**: Verses, hooks, often features/collaborations

## Subgenres & Styles

| Style | Description | Reference Artists |
|-------|-------------|-------------------|
| Boom-Bap | Classic East Coast, sample-heavy, drum breaks | Nas, Wu-Tang Clan, A Tribe Called Quest |
| Trap | 808s, hi-hats, Southern influence | Future, Migos, Travis Scott |
| Conscious | Socially aware, lyrical depth | Kendrick Lamar, J. Cole, Common |
| Documentary | Story-driven, narrative focus | The Notorious B.I.G., Slick Rick |
| Nerdcore | Tech/geek culture themes | MC Frontalot, Dual Core, Mega Ran |
| Lo-Fi Hip-Hop | Chill, jazzy, atmospheric | Nujabes, J Dilla |
