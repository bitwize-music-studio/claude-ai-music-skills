# Eclipse Total

## Concepto
Una canción sobre el momento exacto de un eclipse solar total, usando la oscuridad repentina como metáfora de una revelación emocional inesperada. El track debe transmitir asombro, misterio y una belleza inquietante.

## Style Box
*Copiar esto en el campo "Style of Music" de Suno:*

```
Female alto, ethereal, breathy delivery with growing intensity. Dream pop with shoegaze influences.
Reverb-drenched guitars, shimmering synths, slow-building drums, deep bass.
Atmospheric production, wide stereo field, dynamic crescendo. 75 BPM, D minor.
```

## Exclude Styles (opcional, máx 2-4 items)
*Negative prompts — añadir al final del Style Box cuando pegues en Suno:*

```
no happy vocals
no acoustic guitar
```

## Checklist de Calidad

Antes de marcar como completado:

- [x] Claridad vocal y pronunciación correcta
- [x] Género/estilo coincide con intención (dream pop/shoegaze)
- [x] Tono emocional apropiado (etéreo, misterioso)
- [x] Balance de mezcla (voces presentes pero no dominantes)
- [x] Estructura sigue los tags de sección
- [x] No hay cortes raros o loops
- [x] No hay instrumentos no deseados
- [x] Stems descargados y nombrados correctamente

## Notas de Producción

**Lecciones aprendidas:**
- Poner la descripción vocal PRIMERO funcionó perfectamente
- El rango de 4-7 descriptores es clave - mi primer intento tenía 12 y sonó confuso
- Excluir "acoustic guitar" fue necesario porque Suno tendía a añadirlo
- La estructura con [Guitar Solo] antes del coro final dio el crescendo deseado
