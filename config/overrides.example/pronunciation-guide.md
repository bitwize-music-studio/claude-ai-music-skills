## How to Add Entries

When `/pronunciation-specialist` discovers a new pronunciation issue:

1. Add the word to the appropriate section above
2. Include: standard spelling, phonetic spelling, context notes
3. The pronunciation specialist will auto-add entries here when it discovers issues

## Format Rules

- Use hyphens for syllable breaks: `Sin-ah-lo-ah`
- Use capital letters for stressed syllables if ambiguous: `reh-CORD` (verb) vs `REH-cord` (noun)
- Add notes for context (album name, character, etc.)
