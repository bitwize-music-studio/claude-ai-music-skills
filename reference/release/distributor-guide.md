# Distributor Comparison Guide

Compare digital music distributors for getting your music on streaming platforms.

---

## Distributor Overview

| Distributor | Pricing Model | Best For |
|-------------|---------------|----------|
| **DistroKid** | Annual subscription | Frequent releasers, singles |
| **TuneCore** | Per-release fee | Selective releases, established artists |
| **CD Baby** | One-time fee | Independent artists, long-term catalog |
| **Ditto Music** | Annual subscription | Budget-conscious, international |

| **CD Baby** | $9.95 once | $29-49 once | None | 91% to artist |

## When to Use Each

### DistroKid

**Best for**:
- Frequent releasers (multiple singles/year)
- Experimentation (easy to release, remove, re-upload)
- Budget-conscious with many releases

**Considerations**:
- Music removed if subscription lapses
- Some features cost extra (YouTube Content ID)
- Limited customer support

### TuneCore

**Best for**:
- Established artists with proven catalog
- Selective releases (few but important)
- Need publishing administration

**Considerations**:
- Per-release fees add up
- Annual renewal required
- Higher cost for frequent releasers

### CD Baby

**Best for**:
- Long-term catalog (pay once, stays forever)
- Sync licensing opportunities
- Artists who don't want recurring fees

**Considerations**:
- 9% revenue cut (vs 100% at others)
- Higher upfront cost
- Slower processing

### Ditto Music

**Best for**:
- Budget-conscious unlimited releases
- International artists (UK-based, strong in EU)
- Basic needs without extras

**Considerations**:
- Fewer advanced features
- Less established reputation
- Support quality varies

## Recommended for AI Music Creators

### Primary Recommendation: DistroKid

**Why**:
- Unlimited releases = experiment freely
- Low annual cost
- Quick turnaround
- Easy to remove/re-upload if needed
- Split payments for collaborations

**Tier recommendation**: Musician Plus ($39.99/yr)
- Includes YouTube Content ID
- Leave a Legacy (keeps music up if you stop paying)
- Customizable release date

### Alternative: CD Baby (for select albums)

**Why**:
- One-time fee, no renewal worry
- Music stays live forever
- Good for "finished" albums you won't change

## Cover Song Licensing

If releasing cover songs:

| Distributor | Cover License Service |
|-------------|----------------------|
| DistroKid | Via Easy Song Licensing |
| TuneCore | Via Harry Fox Agency |
| CD Baby | Via Songfile |
| Ditto | Via third-party |

**Cost**: Typically $12-15 per song + royalties per stream

**Note**: AI-generated music using original compositions doesn't need cover licensing.

---

## Switching Distributors

**Can you switch?**
- Yes, but you'll lose stream counts
- Keep old release live during transition
- New ISRC = new song to algorithms

**Best practice**:
- Choose carefully upfront
- Don't switch for minor reasons
- If switching, overlap releases by 2+ weeks
